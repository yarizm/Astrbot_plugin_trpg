# Changelog

## 0.1.0 - 2026-05-09

- Added Markdown scenario import and deterministic scenario splitting.
- Added draft review, publish, scenario list, group selection, current scenario, reset, and export commands.
- Added built-in starter scenarios that can be bootstrapped on first run.
- Added LLM-callable TRPG tools for listing scenarios, group selection, viewing group selection, solo start, solo status, and solo ending.
- Added private-chat solo TRPG mode with LLM-driven narration, dice/notes/progress/session-end tools, summaries, and history persistence.
- Added SQLite persistence for imports, scenario candidates, group sessions, solo sessions, and session history.
- Added AstrBot Dashboard WebUI for scenario management, upload/import, session history, and configuration.
- Added repository release files, CI workflow, ruff configuration, and clearer installation/security documentation.
